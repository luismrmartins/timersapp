## Code

```ts
function nextOccurrence(hour: number, minute: number, now = Date.now()) {
  const d = new Date(now);
  d.setHours(hour, minute, 0, 0);
  if (d.getTime() <= now) d.setDate(d.getDate() + 1);
  return d.getTime();
}
```

That's it — push an `.mdx` file under `content/blog/` and it shows up in the index.
